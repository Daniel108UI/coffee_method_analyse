# Анализ влияния метода обработки кофейного зерна на вкусовые характеристики

## 1. Описание данных

В исследовании использовался датасет Coffee Quality Institute (CQI), содержащий информацию о качестве кофейного зерна, включая метод обработки, ботанический вид, страну и континент происхождения, год урожая, а также экспертную оценку вкуса.

Из анализа были исключены наблюдения с отсутствующим значением метода обработки. Итоговая выборка составила **n = 1169 наблюдений**.

### 1.1. Распределение по методам обработки

В датасете представлены следующие методы обработки кофейного зерна:

- Washed / Wet — **69.7%**
    
- Natural / Dry — **22.1%**
    
- Semi-washed / Semi-pulped — **4.8%**
    
- Other — **2.2%**
    
- Pulped natural / honey — **1.2%**
    

Таким образом, данные характеризуются **сильной несбалансированностью**, с доминированием мытой обработки (Washed / Wet).

### 1.2. Ботанический вид

Распределение по виду кофе:

- Arabica — **99.1%**
    
- Robusta — **0.9%**
    

Датасет практически полностью состоит из кофе сорта Arabica.

### 1.3. Год урожая

Наиболее представленные годы урожая:

- 2012 — 30.4%
    
- 2014 — 21.2%
    
- 2013 — 17.5%
    
- 2015 — 13.3%
    
- 2016 — 8.8%
    
- 2017 — 7.4%
    

Остальные годы представлены незначительно.

### 1.4. Страна происхождения

Наиболее часто встречающиеся страны:

- Mexico — 19.6%
    
- Guatemala — 14.8%
    
- Colombia — 12.7%
    
- Brazil — 10.1%
    
- Taiwan — 6.0%
    
- United States — 5.4%
    
- Honduras — 4.2%
    
- Costa Rica — 4.2%
    

Остальные страны представлены в малых долях (< 3%).

### 1.5. Описательная статистика вкусовой оценки

Для переменной _Flavor_ получены следующие характеристики:

- Среднее значение: **7.51**
    
- Стандартное отклонение: **0.34**
    
- Медиана: **7.5**
    
- Минимум: **6.08**
    
- Максимум: **8.83**
    
- Асимметрия: **–0.33** (слабая левосторонняя)
    
- Эксцесс: **1.73** (умеренная заострённость)
    

Распределение вкусовых оценок близко к нормальному, с умеренной концентрацией значений вокруг среднего.

---

## 2. Проблема несбалансированности данных

Первичный анализ показал, что выборка существенно несбалансирована по методам обработки. В частности, метод Washed / Wet составляет почти 70% наблюдений, тогда как некоторые группы (например, Pulped natural / honey) представлены менее чем 2%.

Такая структура данных может:

- Искажать результаты дисперсионного анализа (ANOVA),
    
- Приводить к завышенной статистической мощности для доминирующих групп,
    
- Уменьшать чувствительность анализа к различиям между малочисленными группами.
    

В связи с этим был применён подход балансировки выборки.

---

## 3. Балансировка выборки

Для получения сопоставимых групп была выполнена стратифицированная случайная выборка с приведением всех групп к размеру наименьшей группы:

- Размер наименьшей группы: **14 наблюдений**
    
- Итоговый сбалансированный датасет:  
    6 методов × 14 наблюдений = **84 наблюдения**
    

Каждая группа методов обработки была представлена равным числом наблюдений (n = 14).

---

## 4. Проверка предпосылок

### 4.1. Гомогенность дисперсий (тест Ливеня)

Для сбалансированной выборки был проведён тест Ливеня:

- F(5, 78) = 0.8548
    
- p = 0.5154
    

Результат не является статистически значимым (p > 0.05), что свидетельствует об отсутствии различий в дисперсиях между группами. Предпосылка гомогенности дисперсий для ANOVA выполнена.

---

## 5. Дисперсионный анализ (ANOVA)

Для оценки влияния метода обработки на вкусовые характеристики был выполнен однофакторный дисперсионный анализ:

- F(5, 78) = 2.071
    
- p = 0.0779
    

Результат находится на уровне статистической тенденции (p < 0.10), но не достигает стандартного уровня значимости p < 0.05. Это указывает на **слабое общее влияние метода обработки на вкусовую оценку** в сбалансированной выборке.

---

## 6. Post hoc анализ (Tukey HSD)

Для парных сравнений между методами обработки был применён тест Tukey HSD.

Статистически значимое различие было обнаружено только для одной пары:

- **Washed / Wet – Other**
    
    - diff = 0.352
        
    - 95% ДИ: [0.0068; 0.6975]
        
    - p = 0.043
        

Для всех остальных пар различия оказались статистически незначимыми (p > 0.05).

---

## 7. Интерпретация результатов

Результаты показывают, что:

1. В исходной (несбалансированной) выборке ранее наблюдались значимые различия между натуральной и мытой обработкой, однако эти различия могли быть частично обусловлены сильной несбалансированностью данных.
    
2. После балансировки выборки:
    
    - Общее влияние метода обработки на вкус ослабляется и становится статистически незначимым на уровне 0.05.
    
    - Единственное устойчивое различие обнаружено между группами **Washed / Wet** и **Other**. То есть статистически значимые вкусовые различия присутствуют только для зерна, мытой обработки, однако группа _Other_ является разнородной и слабо интерпретируемой с технологической точки зрения.
        
3. Различия между ключевыми промышленными методами (Washed, Natural, Semi-washed, Honey) в сбалансированной выборке статистически не подтверждаются.
    

---

## 8. Ограничения исследования

- Очень малые размеры некоторых групп (n = 14) снижают статистическую мощность анализа.

- В анализ не были включены потенциальные ковариаты (страна, год урожая, высота произрастания и др.), которые могут существенно влиять на вкус.


---

## 9. Выводы

В рамках данного исследования не было получено убедительных доказательств того, что метод обработки кофейного зерна оказывает сильное и устойчивое влияние на вкусовую оценку после устранения эффекта несбалансированности данных. Результаты указывают на то, что ранее выявленные различия между методами могут быть частично объяснены структурой выборки, а не только реальными технологическими эффектами.
